# Commands:
!revert: Restore to snapshot
!restartvm: Restart the VM
!shutdown: Shutdown the VM (ACPI shutdown)
!forceshutdown: Shutdown the VM (forcefully)
!move: (may not work, didnt test mouse control commands) Move mouse
!click: (may not work, didnt test mouse control commands) Left click
!rclick: (may not work, didnt test mouse control commands) Right click
!mclick: (may not work, didnt test mouse control commands) Middle click
!startvm: Start the VM
!run: Run a command
!type: Type text
!send: Type text and press enter
!key: Press a key
!combo: Press a key combination
!wait: Wait (maximum 5 seconds)
